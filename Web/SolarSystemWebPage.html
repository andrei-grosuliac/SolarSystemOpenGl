Solar System in OpenGL

By Andrei Grosuliac

April 5, 2023

GitHub

 

 

Project Description

What if our solar system was binary, meaning there was two suns?

How would planets orbit around them and still be habitable?

This OpenGL project aims to create such an immersive experience to help users visualize such a system.

 

Libraries

GLFW, GLEW, GLM

ASSIMP, SOIL2

 

Objectives

1. Create 3D models of the planets and stars

- 3D Models found from the Internet for planet and Stars.

- Loaded with ASSIMP and Model/Mesh classes.

 

2. Use textures and astronomical data to enhance visual fidelity

- 2D Textures found from the Internet for plant and Stars and skybox

- Loaded with SOIL2 and mipmaps generated

                    


                    

3. Implement accurate movement of the planets

- 2 suns orbiting in a circle around their center of mass

- Planets orbiting in an ellipse around the suns’ center of mass

 

 

4. Implement a realistic lighting model for a binary star system

- Local illumination with 2 rotating point lights: 1 white, 1 red

- Use of ambient, diffuse and specular components on planets’ textures

 

 

5. Create an interactive, orbital camera system

W: zoom in

S: zoom out

D/Right: move right

A/Left: move left

Up: move up

Down: move down

P: pause movement

O: show/hide orbital lines

1: view from above

2: unlocks camera

3: view from earth

M: increase orbital speed

N: decrease orbital speed

=: increase camera speed

-: decrease camera speed
